# Monthly Summary App

This app summarizes monthly import/export quantities and weights from Excel files.

---

## 🚀 Features

- ✅ CustomTkinter GUI
- 📊 Weight & Quantity summary
- 📁 Excel output formatting
- 🧩 Preview and merge rows
- 📂 Works with `.xlsx` files (EximRadars format)
